# Certiceipt
Certiceipt creates a tamper-proof record for receipts by using AI to understand them and blockchain to permanently verify them.

## Features
- AI Receipt Understanding
- Smart fraud detection
- Blockchain Certification
- Tamper Detection

## Services and APIs Used
- Gemini
- Solana

## How to Run
In one terminal run:
- cd backend 

In another terminal run:
- cd mobile 

In the backend terminal run:
- npm install 
- npm run dev

In the mobile on run:
- flutter start -d chrome
